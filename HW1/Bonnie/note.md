1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to structure and present content on the web.

2. What is the minimal structure of an HTML5 document?
<!DOCTYPE html> tells the browser to use HTML5
<html> is the root element
<head> contains metadata
<body> contains visible content

3. What is the purpose of the <meta> tag?
The <meta> tag provides metadata about the document,like SEO information, character encoding

4.What is the difference between <head> and <header>?
<head>: Contains metadata and Not displayed on the page
<header>: Contains visible content , displayed on the page

5. What is the <form> tag used for in HTML?
The <form> tag is used to collect user input and submit it to a server.
Like: Login forms, search boxed

6. Explain the following code
<a href="http://example.com/sample_page/" rel="noreferrer nofollow">Link</a>

rel="nofollow":Tells search engines not to follow or rank the link
rel="noreferrer":Prevents sending referrer information

7. How do you serve your page in multiple languages?

8. What are semantic HTML tags and why are they important?
Semantic HTML tags clearly describe their meaning and content.
Examples:<header> <nav> <section> <footer>
They are good to improve accessibility (screen readers) and improve SEO
