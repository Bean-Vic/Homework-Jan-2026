HTML

1. What is HTML?

HTML is the structure and content of a webpage

2. What is the purpose of the <meta> tag?

<meta> tags give extra information about a webpage to browsers and search engines.

3. What is the minimal structure of an HTML5 document?

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Document</title>
  </head>
  <body>
  </body>
</html>

4. What is the difference between <head> and <header> ?

<head> provides information for the browser, while <header> defines the header section of visible content.

5. What element can we use to create a dropdown list?

We use the <select> element with <option> elements to create a dropdown list

6. What is the <form> tag used for in HTML?

The <form> tag is used to collect user input and send the data to a server.

7. Explain the rel="noreferrer nofollow" attribute in <a> tag? How can we open the link in a new tab?

rel="noreferrer" prevents the browser from sending referrer information to the linked page.

rel="nofollow" tells search engines not to pass SEO ranking value to the link.

To open a link in a new tab, use target="\_blank".

8. How do you serve your page in multiple languages?

By using translated content and language switching, usually with different URLs and the lang attribute.

   make the page more logical for human and browser and even screen reader. clearly telling the stucture.
10. What’s the difference between SVG and Canvas?

SVG is vector-based and DOM-based, while Canvas is pixel-based and bitmap-based.
