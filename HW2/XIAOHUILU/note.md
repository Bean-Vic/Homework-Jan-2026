1. What is CSS?

CSS stands for Cascading Style Sheets, and it is used to control the layout, colors, fonts, and visual appearance of HTML elements on a web page.

⸻

2. How do you link a CSS file to an HTML document?

A CSS file is linked to an HTML document by using the <link> tag inside the <head> section and setting the rel attribute to stylesheet and the href attribute to the CSS file path.

⸻

3. What is a block element, and how is it different from inline and inline-block elements?

A block element starts on a new line and takes up the full width of its container, an inline element stays on the same line and does not allow width or height to be set, and an inline-block element stays on the same line but allows width and height to be set.

⸻

4. What is the difference between a pseudo-class and a pseudo-element?

A pseudo-class is used to define a special state of an element, such as hover or focus, while a pseudo-element is used to style a specific part of an element, such as the first line or content before the element.

⸻

5. What is the difference between the child combinator and the descendant combinator?

The child combinator selects only the direct children of an element, while the descendant combinator selects all elements that are nested inside another element at any depth.

⸻

6. What are two ways to make an element invisible, and what is the difference?

An element can be made invisible using display: none, which removes it from the layout, or visibility: hidden, which hides the element but still keeps its space.

⸻

7. What is the Box Model, and what are its parts?

The CSS Box Model describes how elements are structured and consists of content, padding, border, and margin, where content holds the data, padding creates space inside the border, border surrounds the padding, and margin creates space outside the element.

⸻

8. What is the usage of !important, and what are some use cases?

The !important rule is used to force a CSS style to override other rules, and it is mainly used to override third-party styles or for quick debugging.

⸻

9. What does z-index do?

The z-index property controls the stacking order of positioned elements, where elements with a higher z-index appear on top of elements with a lower value.

⸻

10. Can padding and margin be negative?

Margin can have negative values to adjust spacing, but padding cannot be negative in CSS.

⸻

11. How do you center a block element with CSS?

A block element can be centered horizontally by setting its left and right margins to auto, or both horizontally and vertically by using Flexbox with justify-content and align-items set to center.

⸻

12. What are grid items, and what are some grid item properties?

Grid items are the direct children of a grid container, and properties such as grid-column, grid-row, and grid-area are used to control their position and size within the grid.

⸻

13. What is a flex container, and what are some flex container properties?

A flex container is an element with display: flex, and properties such as flex-direction, justify-content, align-items, and flex-wrap are used to control layout and alignment.

⸻

14. If a parent has width 200px and the child width is set to auto, what happens?

When the child element’s width is set to auto, it expands to fill the available width of the parent, which is 200px, and both the left and right margins default to 0 unless explicitly defined.

⸻

15. What is the difference between px, em, and rem in font sizing?

The px unit is a fixed size, em is relative to the font size of the parent element, and rem is relative to the font size of the root element, which makes rem more consistent for responsive design.

⸻

16. What is responsive web design, and how do we achieve it?

Responsive web design is a technique that ensures a website works well on different screen sizes, and it is achieved using media queries, flexible layouts, relative units, and responsive images.
